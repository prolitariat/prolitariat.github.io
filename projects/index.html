Tab 2: Setup Assistant — "Is Pendo installed correctly?"

Framework detection (React, Angular, Vue, Svelte, Next.js), snippet analysis, initialization audit, CSP compatibility with per-directive fixes, CNAME host detection, CMP/GDPR consent platform detection, and metadata validation with PII scanning.

Tab 3: Tools — "Run diagnostics and copy issues"

Launch Pendo Debugger (on-page overlay for alerts, config, events, and guides), one-click Validate Install and Validate Environment console commands, plus Copy Issues to Clipboard — a priority-sorted plain-English report with remediation suggestions, ready to paste into Slack or Jira.

Key Features

♦Live Service Status — Auto-detects realm (US/EU/US1/JP), shows real-time Pendo service health
♦QA Test Harness — 8 preset scenarios for regression testing without a live Pendo installation
♦First-Run Tour — 5-step guided onboarding with spotlight overlays and keyboard navigation
♦WCAG 2.1 AA — Full keyboard accessibility, ARIA roles, focus-visible rings, 51 color combos passing
♦PII-Scrubbed Feedback — Emails, phone numbers, SSNs, credit cards, and IPs stripped before submission
♦Zero Dependencies — Pure vanilla JS, no build step, no npm, no bundler

Analytics Dashboard

A companion analytics dashboard tracks extension usage in real time — total events, Pendo detection rates, average health scores, version distribution, and event breakdowns — all powered by a Cloudflare Workers backend. No PII, no cookies.
